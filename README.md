# 💰 Expense Tracker System (MERN)

A **full-stack, interactive Expense Tracker** built with **MongoDB, Express, React, and Node.js (MERN)**.  
Track income and expenses in real-time with a **responsive, animated, and user-friendly interface**.

## ✨ Features

- ✅ Add, edit, and delete **income and expense transactions** dynamically  
- ✅ Real-time calculation of **total income, total expenses, and balance**  
- ✅ **Smooth CSS animations** for adding/deleting transactions  
- ✅ **Color-coded indicators** for income (green) and expenses (red)  
- ✅ **User authentication** for secure financial management  
- ✅ Fully **responsive design** for mobile and desktop  
- ✅ **RESTful API integration** with Node.js and Express
