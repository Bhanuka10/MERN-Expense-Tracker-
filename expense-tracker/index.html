Expense Tracker Application

Welcome to the Expense Tracker application! This is a full-stack MERN (MongoDB, Express, React, Node.js) application for tracking personal expenses.

Getting Started

Follow these steps to run the application:

  1. Make sure MongoDB is installed and running on your machine
  2. Start the backend server:
    Run Backend
  3. Start the frontend:
    Run Frontend
  4. Access the application:
    Open Application

Troubleshooting

If you encounter any issues:

  • Check if MongoDB is running:
    Check MongoDB Status
  • Make sure both backend and frontend servers are running
  • Check the terminal for any error messages

Features

  • Add, edit, and delete expenses
  • Categorize expenses
  • View expense statistics and charts
  • Filter and sort expenses
  • Responsive design
